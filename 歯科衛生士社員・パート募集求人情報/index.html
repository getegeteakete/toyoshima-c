あなたの笑顔が患者さんの未来を変える

六本松駅から徒歩3分にある当クリニックは、矯正歯科と小児歯科を専門に、幅広い世代の患者さんの健康を支えるアットホームな歯科医院です。患者さん一人ひとりと長く丁寧に向き合うことを大切にしています。

✓

残業ほぼなし・週休2日

プライベートを大切にできる働きやすい環境。月曜日は午後からの勤務で午前中はゆっくり。

✓

社員特典でスキルアップ

矯正治療・ホワイトニングなど社割制度あり。スタッフ自身も美しく健康に！

✓

安心のサポート体制

矯正未経験でも、20〜40代の先輩スタッフとドクターが丁寧に指導します。

✓

家族・子育てに優しい

お子様の学校行事や急な体調不良も気兼ねなくお休みOK。家族手当・家賃補助制度あり（条件あり）。

✓

健康サポート充実

インフルエンザ・肝炎ワクチン補助など健康管理もしっかりサポート。

✓

アクセス抜群

地下鉄七隈線六本松駅徒歩3分。駐車場完備で車通勤もOK。
